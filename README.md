# scRNA-seq Analysis of *Trypanosoma cruzi* Populations

This repository contains the code used to analyze single-cell RNA sequencing data from *Trypanosoma cruzi* as part of our manuscript:

**"Single-cell RNA-seq reveals trans-sialidase-like superfamily gene expression heterogeneity in *Trypanosoma cruzi* populations"**  
Lucas Inchausti, Lucía Bilbao, Vanina A. Campo, Joaquín Garat, José Sotelo-Silveira, Gabriel Rinaldi, Virginia M. Howick, María Ana Duhagon, Javier G. De Gaudenzi\*, Pablo Smircich\*  
[2025]

We performed single-cell RNA sequencing (scRNA-seq) of a mixed population of amastigotes and cell-derived trypomastigotes of *Trypanosoma cruzi* strain Dm28c. Using 10X Chromium technology, we profiled 3,192 single-cell transcriptomes and identified stage-specific gene expression signatures. Our analysis revealed that members of the trans-sialidase-like (TcS) superfamily are expressed with significantly greater heterogeneity than single-copy genes, with each individual trypomastigote displaying a unique TcS expression profile and no recurrent combinations observed across cells.

🔹 Note: This repository contains only the R code used for data processing, clustering, and figure generation. Raw sequencing data are available in NCBI SRA under BioProject [PRJNA1200704](https://www.ncbi.nlm.nih.gov/bioproject/PRJNA1200704).

## 📂 Input Files
```text
- cells_x_genes.mtx              -> Sparse count matrix, replicate 1
- cells_x_genes.genes.txt        -> Gene IDs, replicate 1
- cells_x_genes.barcodes.txt     -> Cell barcodes, replicate 1
- cells_x_genes_2.mtx            -> Sparse count matrix, replicate 2
- cells_x_genes.genes_2.txt      -> Gene IDs, replicate 2
- cells_x_genes.barcodes_2.txt   -> Cell barcodes, replicate 2
- anotacion_dm28c_con_mt.gff     -> T. cruzi Dm28c genome annotation (TriTrypDB release 62 + kDNA)
- Ribsoomal_RNA.csv              -> List of ribosomal RNA gene IDs to exclude
```

> Count matrices are generated by pseudoalignment with `kallisto bustools` against the *T. cruzi* Dm28c 2018 genome (TriTrypDB release 62), with 3′UTR regions extended using `peaks2UTR`.

## 📈 Output Highlights
```text
- QC plots: UMI, gene count, and mitochondrial ratio distributions per cell.
- UMAP: 2D visualization of cell clusters (amastigotes, trypomastigotes, transitioning cells).
- Marker heatmap: top 10 upregulated genes per cluster.
- Gini index plots: expression heterogeneity across gene categories.
- TcS heatmaps: per-cell trans-sialidase expression profiles.
- Cumulative expression plots: equal contribution of TcS members per cell.
- Raincloud / boxplot: genomic context of high- vs. low-detection TcS genes.
```
